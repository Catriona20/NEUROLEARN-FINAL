I have completely revamped the learning session tasks to focus on **literacy fundamentals** in a gamified way, replacing the complex Harry Potter trivia with accessible "monkey typing" and reading exercises.

### 1. **New Task Structure**
Every level now consists of 3 simple, foundational tasks:
1.  **Type**: Monkey typing practice (Input strictly validated).
2.  **Read**: Sight word reading.
3.  **Say**: Pronunciation practice.

### 2. **Gamified Typing Interaction**
- **Added Input Field**: When a task starts with "Type:", a beautiful, game-styled text field appears.
- **Validation**: Users *must* type the word correctly to proceed.
- **Feedback**: If the word is wrong, a "Keep trying!" message appears.
- **Success**: Correct typing triggers the "I Did It!" / Success state with confetti.

### 3. **Simplified Content (Levels 1-12)**
The content has been reset to progressive CVC (Consonant-Vowel-Consonant) and common sight words, making it much friendlier for the target audience.
