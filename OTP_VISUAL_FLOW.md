## ğŸ¯ User Interaction Points

### 1. Email Input
```
Action: User types email
Validation: Email format check
Feedback: Error text if invalid
```

### 2. Send OTP Button
```
Action: Click button
Trigger: sendOTP() API call
Loading: Show spinner
Success: Animate OTP input in
Error: Show error message
```

### 3. OTP Input Boxes
```
Action: User types digit
Behavior: Auto-focus next box
Complete: Auto-verify when 6 digits entered
Paste: Auto-fill all boxes
```

### 4. Resend OTP Button
```
Action: Click button
Trigger: Hide OTP input, re-enable email
Result: Can send new OTP
```

### 5. Verify OTP (Auto)
```
Trigger: 6 digits entered
API Call: verifyOTP()
Success: Navigate to next screen
Error: Shake animation + clear boxes
```
